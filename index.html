A small and light weight Login Form



HowTo:

To create this widget, just write a div block and put these code below in you script tag
                1 <div class='id'>Admin</div>
            
                1 $(function(){
                2     $('.login').PopUpLoginForm('login',submit,login,reg);
                3     // It has four arguments 
                4     //first arg is a string, means the form is a 
                5     //login form or it's a form that 
                6     // someone has already logged in ('loged')
                7     // The orthers are callback function, each of them means the
                8     // function to load when events triggered
                9 })
